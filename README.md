## Español

Página personal de servicios freelance — desarrollo web fullstack con especialización en contabilidad, finanzas e integración de IA para pymes.

### Sobre el proyecto
Landing page construida con Next.js, TypeScript y Tailwind CSS. Incluye secciones de presentación, servicios, portfolio, sobre mí y contacto.

### Servicios
- Desarrollo de aplicaciones web fullstack
- Diseño y mantenimiento de páginas web
- Chatbots de IA para atención al cliente
- Dashboards y herramientas de analítica para pymes

### Stack técnico
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- OpenAI API

### Instalación local
\`\`\`bash
git clone https://github.com/MMMazaira/marianela-dev.git
cd marianela-dev
npm install
